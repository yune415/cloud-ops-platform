# 勒索病毒应急响应预案

**生效时间**: 2026-05-19  
**适用对象**: 云原生运维平台所有节点  

## 一、检测
- 发现异常：文件后缀异常、勒索信、CPU/磁盘突增
- 确认后：立即断网（安全组入方向置空）
- 上报：钉钉告警 + 人工通知

## 二、隔离
- 停止 MySQL、Nginx、Docker 容器
- 快照：对云盘做手动快照（阿里云控制台）

## 三、恢复（PITR）
- 从异地 OSS 拉取最新备份
- 执行 PITR 恢复（恢复到感染前 1 小时）
- 验证完整性

## 四、根因分析与复盘
- 分析入侵路径（日志审计）
- 加固弱点（补丁、凭证轮换）

**演练频率**: 每季度一次，首次演练定于 2026-06-01
